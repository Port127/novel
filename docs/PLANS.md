# PLANS

本仓库的执行计划索引。

## 工作流程
1. 在 `docs/exec-plans/active/` 创建计划文件。
2. 执行并验证。
3. 完成后移至 `docs/exec-plans/completed/`。
4. 记录后续技术债在 `docs/exec-plans/tech-debt-tracker.md`。

## 计划索引
- 进行中的计划：`docs/exec-plans/active/README.md`
- 已完成的计划：`docs/exec-plans/completed/README.md`
- 技术债跟踪：`docs/exec-plans/tech-debt-tracker.md`
