# 章节定位与张弛 (Chapter Functions)

> **用途**：Phase 2 和 Phase 3 用于合理分配每一章的压力级别，避免“每章像短篇”导致的读者疲劳。

一卷里章的压力本就该有高有低。这里的压力级指**爆发 / 冲突烈度**（复用悬念刻度 1微 / 2小 / 3中 / 4大 / 5极），不是情感烈度——关系回收 / 泪目这类章可以压力低（爆发小）但情感强度顶满。

| 章节定位 | 功能 | 压力级 | 钩子要求 | 密点要求 |
|------|------|--------|----------|----------|
| **高压章** | 打脸 / 掉马 / 反转 / 高潮；关键情节落点 | 4-5 | 章尾强钩子 | 依赖密点展开（≥250字），多个密点 |
| **推进章** | 主线推进、冲突升级、单元剧中段 | 2-3 | 章尾中钩子 | 必须有密点推进主线 |
| **修炼试错章** | 实力 / 资源积累、能力铺垫、失败试错 | 2 | 进度钩子（差一点 / 即将突破） | 允许较少密点，多铺垫 |
| **关系回收章** | 感情线兑现、关系转折、身边人日常 | 2-3 | 情感钩子（拉扯） | 需 ≥1 个情感密点（情感爆发） |
| **低压生活章** | 高潮后喘息、关系升温、张弛调节 | 1-2 | 允许弱钩子或仅情绪钩子 | 允许 0 密点，靠疏点/铺垫凑字 |
| **信息整理章** | 转场、设定交代、时间缝合 | 1 | 结尾给下一段目标 | 允许 0 密点，尽量避免单章成篇 |

### 两条底线规则

1. **期待不断、爆发分布**。
   - 低压生活章可以没有爽点和反转，但每章都要给读者一个往下看的理由（微好奇、阶段目标、暧昧期待）。
   - 掉的是爆发烈度，不是往前的拉力。
2. **禁情绪母题扎堆**。
   - 比“没有起伏”更隐蔽的疲劳源，是连续多章同一情绪母题高位运行（例如连续五章家国大义/热血/泪目）。
   - 必须通过换情绪、换钩子方式、多线穿插来差异化，不能情绪趋同。

### 与细纲「字数预算」的衔接

- 章节定位只改变密/疏配比的下限，**不改字数预算契约**。
- 高压章靠多密点（每个 ≥250 字）达到总目标。
- 低压生活 / 信息整理章靠铺垫（120-150字）与疏点（≈40字）凑字，允许 0 密点。
- 关系回收章必须保底有一个情感密点。
- 如果实际分配字数不足，高压章应当增加密点或进一步拆分爽点；低压章应当增加铺垫、互动，而非强塞爽点。
