# 章节摘要模板

> **用途**：Phase 3 生成章节摘要时对照使用。
> **核心结构**：五要素摘要 + 多线情节 + 出场人物 + 字数预算。

---

## 章节摘要结构

每章摘要包含以下字段：

```yaml
chapter:
  number: 1
  title: "第一章 开始"
  density: 密                   # 密/中/疏
  tension: 3                    # 1-5
  words_target: 3000
  summary:
    main_plot: ""               # 主线推进
    character_change: ""        # 人物变化
    foreshadowing: ""           # 伏笔（埋设/回收）
    emotion: ""                 # 情绪基调
    hook: ""                    # 章末钩子
  plotlines:
    main: ""
    sub: ""                     # 可选
    hidden: ""                  # 可选
  characters:
    - name: ""
      role: protagonist
      function: ""
  beats: []
```

---

## 五要素摘要

### 1. main_plot（主线推进）
**定义**：本章在主线剧情上推进了什么？
**写法**：一句话概括，15-50 字。

### 2. character_change（人物变化）
**定义**：本章中主角/重要角色发生了什么内在变化？
**写法**：从 A 状态变为 B 状态，20-60 字。

### 3. foreshadowing（伏笔）
**定义**：本章埋设/回收了什么伏笔？
**写法**：标注"埋设"或"回收"，说明伏笔内容。

### 4. emotion（情绪基调）
**定义**：本章的主导情绪是什么？
**写法**：1-2 个情绪词 + 强度（1-5）。

### 5. hook（章末钩子）
**定义**：章末留给读者的悬念/期待是什么？
**写法**：一句话描述，让读者想知道"然后呢？"

---

## 多线情节标注

| 线类型 | 定义 | 推进频率 |
|--------|------|---------|
| 主线（main） | 推动核心剧情 | 每章必须 |
| 副线（sub） | 感情线、配角线 | 可隔章推进 |
| 暗线（hidden） | 尚未完全察觉的线索 | 通过伏笔暗示 |

---

## 字数预算

| 密度 | 目标字数 | 适用范围 |
|------|---------|---------|
| 极密 | 4000-5000 | 高潮/大战 |
| 密 | 3000-4000 | 冲突/转折 |
| 中 | 2500-3500 | 标准章节 |
| 疏 | 2000-2500 | 过渡/铺垫 |

---

## 质量检查

| # | 检查项 | 通过标准 |
|---|--------|----------|
| 1 | 五要素完整 | 每章都有 main_plot/hook |
| 2 | 字数合理 | 2000-5000 字 |
| 3 | 人物明确 | 出场人物列表不为空 |
| 4 | 钩子存在 | 每章末尾有章末钩子 |
| 5 | 主线推进 | main_plot 不为空 |
