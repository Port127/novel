# 黄金三章核心规则

> **用途**：Phase 1-4 锻造黄金三章时对照使用。
> **核心**：前三章决定读者去留，必须满足四个硬性要求。

---

## 四大硬性要求

| 要求 | 位置 | 标准 | 不达标后果 |
|------|------|------|-----------|
| 首冲突 | 第1章前300字 | 必须有冲突/悬念/钩子 | 读者弃书 |
| 人设建立 | 第1章 | 读者知道主角是谁、处境如何 | 无法代入 |
| 金手指亮相 | 第2章 | 展示核心优势/能力 | 读者不知"爽点"在哪 |
| 首个小高潮 | 第3章 | 让读者看到"爽"的可能性 | 读者不知"为什么要追" |

---

## 第1章规则

### 前300字必须完成

1. **建立场景**：读者知道"这是哪"
2. **展示主角**：读者知道"这是谁"
3. **制造冲突**：读者知道"发生了什么事"

### 冲突类型（按品类）

| 品类 | 冲突类型 | 示例 |
|------|---------|------|
| 玄幻 | 被欺辱/被退婚 | "一个废物也配修炼？" |
| 都市重生 | 重生节点确认 | 睁开眼看到2009年的日历 |
| 言情 | 尴尬相遇 | 撞见男主换衣服 |
| 系统 | 系统激活 | "叮——系统绑定中..." |
| 悬疑 | 异常事件 | 发现尸体上的奇怪符号 |

### 人设建立技巧

- **展示而非告知**：通过行为展示性格，不要直接写"他很勇敢"
- **制造共情**：让读者同情/认同主角
- **留下悬念**：主角有什么秘密/特殊之处？

---

## 第2章规则

### 核心任务：展示金手指/核心优势

| 品类 | 展示内容 | 技巧 |
|------|---------|------|
| 玄幻 | 金手指觉醒/传承 | 意外触发，展示规则 |
| 都市重生 | 先知优势 | 利用信息差获取第一个好处 |
| 言情 | 化学反应 | 男女主再次相遇，产生火花 |
| 系统 | 系统机制 | 首个任务，展示奖励 |

### 注意事项

- **有限制**：金手指不能无敌，要有代价/冷却/条件
- **有惊喜**：给读者"原来还能这样"的感觉
- **不解释太多**：通过行动展示规则，不要大段说明

---

## 第3章规则

### 核心任务：首个小高潮

**定义**：让读者第一次感受到"爽"，建立"追下去能看到更多爽点"的预期。

### 高潮设计

| 要素 | 说明 |
|------|------|
| 对手 | 一个可以被击败的小反派/障碍 |
| 代价 | 主角付出一定代价（不是一帆风顺） |
| 碾压 | 展示金手指的威力（让读者期待更多） |
| 余波 | 胜利带来的好处+新的悬念 |

### 常见套路

- **打脸**：之前看不起主角的人被打脸
- **逆转**：看似必输的局面被翻盘
- **揭秘**：主角的特殊身份/能力被部分曝光
- **收获**：获得资源/地位/认可

---

## 节奏控制

### 三章节奏模板

```
第1章：冲突(4) → 应对(3) → 悬念(3)
第2章：探索(2) → 发现(3) → 展示(4)
第3章：准备(3) → 对决(5) → 收获(4)
```

### 字数分配

| 章节 | 字数 | 内容密度 |
|------|------|---------|
| 第1章 | 3000-4000 | 快速切入，不拖沓 |
| 第2章 | 3000-4000 | 展示机制，留悬念 |
| 第3章 | 4000-5000 | 高潮要猛，收束要干净 |

| # | 检查项 | 通过标准 |

| 1 | 首冲突 | 第1章前300字内有冲突/钩子 |

| 3 | 金手指 | 第2章展示核心优势 |
| 4 | 小高潮 | 第3章有可感知的"爽点" |
