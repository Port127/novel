# 品类黄金章节拍模板

> **用途**：Phase 1 加载品类对应的黄金三章节拍模板。

---

## 玄幻/仙侠

### 标准套路
废柴受辱 → 金手指觉醒 → 首次反击

### 节拍模板

**第1章：废柴登场**
1. [环境] 宗门/家族场景
2. [冲突] 被欺辱/嘲笑
3. [情绪] 主角的隐忍/愤怒
4. [伏笔] 特殊物品/体质暗示
5. [钩子] 意外触发金手指的征兆

**第2章：金手指觉醒**
1. [揭示] 金手指正式激活
2. [探索] 了解金手指规则
3. [惊喜] 发现金手指的强大
4. [限制] 展示金手指的代价/冷却
5. [钩子] 决定利用金手指改变处境

**第3章：首次反击**
1. [准备] 利用金手指提升实力
2. [冲突] 再次遇到欺辱者
3. [高潮] 碾压式反击
4. [震惊] 围观者的反应
5. [收获] 获得认可/资源

## 都市重生

### 标准套路
重生节点 → 利用先知优势 → 第一步逆袭

### 节拍模板

**第1章：重生确认**

2. [确认] 验证重生事实
3. [情绪] 震惊→接受→决心
4. [规划] 回忆关键事件节点

**第2章：先知优势**

2. [展示] 利用信息差获得好处
3. [惊喜] 周围人的震惊

5. [钩子] 更大的机会/挑战出现

**第3章：第一步逆袭**

2. [冲突] 遇到阻碍/竞争者

4. [收获] 获得第一桶金/认可

## 悬疑

### 标准套路
异常事件 → 调查深入 → 更大谜团

### 节拍模板

**第1章：异常降临**
1. [日常] 平静的日常生活
2. [异常] 发现不寻常的事件/物品
3. [困惑] 无法解释的现象
4. [决定] 决定调查真相
5. [钩子] 发现更深的疑点

**第2章：深入调查**
1. [线索] 找到第一条关键线索
2. [推理] 初步推理分析
3. [阻碍] 调查遇到阻力
4. [发现] 意外发现新的线索
5. [钩子] 线索指向更大的谜团

**第3章：真相初现**
1. [整合] 整合已有线索
2. [突破] 关键推理突破
3. [震惊] 真相的一角被揭开
4. [危险] 意识到危险的接近
5. [钩子] 更大的阴谋浮出水面

---

## 模板使用指南

1. **灵活调整**：模板是参考，不是死规则
2. **品类混搭**：可以融合多个品类的元素
3. **创新空间**：在套路基础上加入创新点
4. **读者预期**：满足品类读者的核心期待
