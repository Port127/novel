# 微节拍设计指南

> **用途**：Phase 2-4 逐段生成正文时对照使用。
> **核心概念**：微节拍是段落级别的节奏单位，控制读者的阅读体验。

---

## 什么是微节拍

**节拍（Beat）**：章节级别的剧情推进单位
**微节拍（Micro-Beat）**：段落/句子级别的信息变化单位

一个微节拍 = 一次信息变化/情绪转折/动作转换

---

## 微节拍类型

| 类型 | 定义 | 示例 | 阅读体验 |
|------|------|------|---------|
| 信息微节拍 | 新信息出现 | "他发现门没锁" | 好奇 |
| 动作微节拍 | 角色行动 | "他推开门" | 紧张 |
| 情绪微节拍 | 情绪变化 | "他感到一阵寒意" | 共情 |
| 对话微节拍 | 信息交换 | "你知道发生了什么吗？" | 期待 |
| 转折微节拍 | 预期被打破 | "但房间里空无一人" | 惊讶 |
| 氛围微节拍 | 环境/节奏变化 | "风突然停了" | 不安 |

---

## 微节拍密度

### 高密度（快节奏）
- 每段都有信息变化
- 段落短（1-3句）
- 适合：冲突、战斗、高潮

**示例**：
```
他冲进去。
 empty。
没人。
但桌上有东西——
一封信。
他的名字。
```

### 中密度（标准节奏）
- 每2-3段一个信息变化
- 段落中等（3-5句）
- 适合：日常推进、对话、准备

### 低密度（慢节奏）
- 每4+段一个信息变化
- 段落长（5+句）
- 适合：氛围营造、内心独白、描写

---

## 微节拍编排规则

### 规则1：变化产生节奏
连续相同类型的微节拍会让读者疲劳。

**错误**：
```
他走路。他看到树。他听到鸟叫。他感到风。他继续走。
```
（全是动作/感知微节拍，无变化）

**正确**：
```
他走路。（动作）
树上有东西在动。（信息）
他停下脚步。（动作）
心跳加速。（情绪）
"是谁？"（对话）
```

### 规则2： tension 起伏
微节拍应该形成小的张力起伏。

```
平静 → 发现 → 紧张 → 释然 → 新发现
```

### 规则3：段落长度变化
长段落和短段落交替，形成视觉节奏。

```
长段落描述场景，建立氛围。

短句。
转折。

长段落展开动作。

一个字。
```

## 黄金三章的微节拍要求

### 第1章前300字
- 必须有至少3个微节拍变化
- 建议密度：中→高
- 必须包含：信息微节拍（建立场景）+ 冲突微节拍

### 第2章金手指展示
- 微节拍要清晰展示规则
- 建议密度：中
- 必须包含：信息微节拍（揭示）+ 情绪微节拍（惊讶/兴奋）

### 第3章高潮
- 微节拍密度最高
- 短段落、快节奏
- 必须包含：动作微节拍 + 转折微节拍

## 去AI味的微节拍技巧

AI写作的微节拍特征：
- 过于规整（每段长度相近）
- 缺乏情绪起伏
- 转折生硬

人工写作的微节拍特征：
- 段落长度不规则
- 情绪有自然起伏
- 转折有铺垫

### 技巧1：插入"无用"细节
真实的写作会有看似无用但增加真实感的细节。

**AI**：
```
他走进房间，看到桌上有一封信。
```

**人工**：
```
他走进房间。
桌上的灰尘很厚，但信是新的。
```

### 技巧2：情绪不直接说
不要直接写"他很生气"，通过动作/细节展示。

**AI**：
```
他感到很愤怒。
```

**人工**：
```
他的手捏紧了。
指节发白。
```

### 技巧3：节奏有呼吸
即使是高潮段落，也要有短暂的"呼吸"。

```
连续紧张动作。
更紧张的动作。

停顿。
一句话的喘息。

继续紧张。
```

## 质量检查

| # | 检查项 | 通过标准 |
|---|--------|----------|
| 1 | 微节拍密度 | 与章节类型匹配 |
| 2 | 类型变化 | 不连续5+同类型微节拍 |
| 3 | 段落长度 | 有长有短，不规整 |
| 4 | 情绪起伏 | 有自然的张力波动 |
| 5 | 无AI痕迹 | 通过 anti-ai-writing 检查 |
