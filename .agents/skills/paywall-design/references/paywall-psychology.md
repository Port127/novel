# 付费心理分析

> **用途**：Phase 2 评估切点时对照使用，分析读者付费决策心理。

---

## 读者付费决策模型

读者决定是否付费时，经历以下心理过程：

```
当前满足感 → 预期满足感 → 成本评估 → 决策
```

### 1. 当前满足感

**定义**：读者在阅读免费章节后获得的满足程度。

**影响因素**：
- 爽点强度（打脸/逆袭/揭秘）
- 人物吸引力（主角/CP）
- 剧情投入度（是否关心后续）

### 2. 预期满足感

**定义**：读者预期付费后能获得的满足程度。

**影响因素**：
- 悬念强度（"然后呢"的好奇）
- 承诺暗示（作者暗示后续更精彩）
- 沉没成本（已经读了这么多，不看可惜）

### 3. 成本评估

**定义**：读者评估付费的成本。

**影响因素**：
- 绝对价格（几块钱）
- 相对价格（值不值）
- 替代选择（有没有其他书可看）

---

## 关键心理机制

### 损失厌恶

**定义**：人们对损失的敏感程度大于对收益的敏感程度。

**应用**：
- 让读者感觉"不付费就看不到精彩的后续"
- 制造"已经投入这么多时间/情感，放弃太可惜"的感觉

### 沉没成本

**定义**：已经付出且不可回收的成本影响决策。

**应用**：
- 免费章节越多，读者投入越多，越难放弃
- 但免费太长会降低付费意愿（边际递减）

### 承诺一致性

**定义**：人们倾向于与自己之前的行为保持一致。

**应用**：
- 读者已经认同主角，会继续关注主角的命运
- 读者已经相信这个故事，会继续追下去

## 切点心理评估

### 最佳切点心理状态

| 心理状态 | 描述 | 如何制造 |
|---------|------|---------|
| 满足 | 刚获得爽点 | 前章设计打脸/逆袭 |
| 好奇 | 想知道后续 | 抛出强悬念 |
| 期待 | 相信后续更精彩 | 暗示更大的冲突 |
| 不舍 | 不想离开这个故事 | 人物吸引力 |

### 最差切点心理状态

| 心理状态 | 描述 | 如何避免 |
|---------|------|---------|
| 疲惫 | 连续高强度阅读 | 切点前安排喘息 |
| 失望 | 期待落空 | 不要在低谷切 |
| 困惑 | 不明白发生了什么 | 确保逻辑清晰 |
| 满足（过度） | 故事告一段落 | 不要在收束章切 |
