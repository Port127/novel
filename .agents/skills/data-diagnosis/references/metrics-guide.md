# 指标体系

> **用途**：Phase 2 计算指标时对照使用。

---

## 核心指标

### 追读率（留存率）

**定义**：读完第 N 章的读者中，继续读第 N+1 章的比例。

**计算**：
```
追读率 = 第N+1章阅读人数 / 第N章阅读人数 × 100%
```

**健康标准**：
| 评级 | 追读率 | 说明 |
|------|--------|------|
| 优秀 | ≥ 75% | 读者粘性强 |
| 良好 | 60-75% | 正常水平 |
| 一般 | 45-60% | 需要改进 |
| 危险 | < 45% | 严重流失 |

---

### 完读率

**定义**：开始读某章的读者中，读完该章的比例。

**计算**：
```
完读率 = 读完人数 / 开始读人数 × 100%
```

**健康标准**：
| 评级 | 完读率 | 说明 |
|------|--------|------|
| 优秀 | ≥ 80% | 章节吸引力强 |
| 良好 | 65-80% | 正常水平 |
| 一般 | 50-65% | 可能有问题 |
| 危险 | < 50% | 章节质量差 |

---

### 互动率

**定义**：阅读章节后进行互动（评论/点赞/收藏）的比例。

**计算**：
```
互动率 = 互动人数 / 阅读人数 × 100%
```

**健康标准**：
| 评级 | 互动率 | 说明 |
|------|--------|------|
| 优秀 | ≥ 5% | 读者参与度高 |
| 良好 | 2-5% | 正常水平 |
| 一般 | 1-2% | 偏低 |
| 危险 | < 1% | 几乎没有互动 |

---

### 打赏率

**定义**：阅读章节后进行打赏的比例。

**计算**：
```
打赏率 = 打赏人数 / 阅读人数 × 100%
```

**说明**：打赏率通常很低（<0.5%），但能反映核心粉丝的付费意愿。

---

## 趋势指标

### 滑动平均

**定义**：过去 N 章的平均值，用于平滑波动。

**用途**：识别趋势，避免单章异常干扰。

**建议窗口**：
- 短期趋势：5 章滑动平均
- 中期趋势：10 章滑动平均
- 长期趋势：20 章滑动平均

### 异常点标记

**定义**：与滑动平均偏差超过阈值的章节。

**阈值**：
- 追读率：偏差 > 15%
- 完读率：偏差 > 20%
- 互动率：偏差 > 50%

---

## 平台差异

不同平台的指标标准可能不同：

| 平台 | 追读率基准 | 完读率基准 | 说明 |
|------|-----------|-----------|------|
| 番茄 | 60% | 70% | 免费模式，读者忠诚度低 |
| 起点 | 70% | 75% | 付费模式，读者质量高 |
| 晋江 | 65% | 75% | 女性读者，粘性强 |
