## 问题汇总

### 🔴 严重问题 (3)

1. **_protocols/ 12 个协议文件未被任何 skill 显式引用**
   - 位置: `.claude/skills/_protocols/`
   - 问题: 11/12 个协议文件未被任何 SKILL.md 引用
   - 影响: 协议无法确保被 agent 遵循，存在"有规则无约束"的风险
   - 建议: 在相关 skill 的 SKILL.md 中显式引用对应协议，或在 pipeline skill 中调用协议规则

2. **consistency-check 与 project-lint 的边界对外不够清晰**
   - 位置: consistency-check/SKILL.md, project-lint/SKILL.md
   - 问题: 两者都声称做"一致性检查"，用户难以区分何时用哪个
   - 影响: 用户可能重复运行或遗漏必要检查
   - 建议: 在 description 中明确区分：lint = 文件/索引/命名机械化检查，consistency = 语义/逻辑/叙事一致性

3. **所有 skill 缺少版本管理和变更记录**
   - 位置: 全部 65 个 skill
   - 问题: 无 version 字段、无变更日志
   - 影响: skill 修改后无法追溯，也无法判断与其他 skill 的兼容性
   - 建议: 至少在 frontmatter 中添加 version 字段，维护 CHANGELOG

### 🟡 警告 (5)

1. **7 个 skill 缺少 argument-hint**
   - 涉及: novel-doctor, timeline-check, relationship-check, novel-list, novel-status, style-list, consistency-check
