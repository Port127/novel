# AGENTS

本仓库是 **Skill Package**（非代码仓库），使用精简 harness。

## 项目标识
- 名称：AI小说写作系统
- 类型：Skill Package
- 用途：面向长篇连载与多项目管理的写作工作台。

## 精简 harness 说明

本项目没有可执行代码、测试套件或 CI 流水线。完整 harness（RELIABILITY、SECURITY、QUALITY_SCORE、evals、exec-plans 等）不适用，已清理。保留的 harness 文件仅限导航和架构契约。
