## 真实前提

一个普通修行者（赵宋）在灵气复苏的浪潮中无声无息地活了80年，最终跨过临界点觉醒了神体。他的神体——而非他本人——穿越回了过去，以独立存在的方式附着在年轻的自己身旁。年轻的赵宋对此一无所知，只知道自己突然觉醒了某种异常能力。

故事的核心引擎是 **三层递进的误读**：
- 世人以为他是超凡者 → 实际是概念者
- 世人以为他是概念者 → 实际是时间锚定/未来回溯
- 赵宋以为这是自己的力量 → 实际是未来的自己留下的遗产

而在他意识深处，未来的自己作为一个逐渐消散的"幽灵"，正在通过有限的信息窗口引导——甚至诱骗——年轻的自己走向某条道路。

---

## 独特机制

### 核心装置：神体-宿主的共生悖论

- **神体本质**：概念核心的一种表现形式，是另一个宇宙（B宇宙）的形态。人体 = 本宇宙（A宇宙）形态。神体 + 人体 = 完整个体。倾向任何一方都有无穷隐患。
- **未来赵宋的存在方式**：独立存在，寄附于神体之上。以潜意识形态驻留在年轻赵宋脑海中。激发神体时可进行有限对话。
- **信息限制**：非不愿说，而是有不可名状的限制。赵宋被B宇宙污染的程度越深，可言说的范围越大。
- **权力关系**：主导权在年轻赵宋手中。未来赵宋只能引导/诱骗，无法强制。
- **消亡机制**：神体消耗 → 未来赵宋逐渐获得更完整的神体控制权 → 完全消散 → 神体完全归属年轻赵宋。
- **终极隐患**：纯人体或纯神体都不是正确答案——双宇宙的平衡才是关键。

### 三层误读的渐进结构

世界对赵宋的认知始终落后一个身位：
1. **超凡者阶段**：世人认为他是空间系超凡者，但他实际拥有概念级能力
2. **概念者阶段**：当世人开始认识到概念的存在并认定他是概念者时，他的底牌已经跃升到"时间锚定/未来回溯"
3. 赵宋始终领先世界认知一个身位——但他自己也不完全理解自己的力量

## 主角解读

### 赵宋（现在·年轻态）

- **性格**：刚毅、稚嫩、坚强、友善但带戒心、有些小冲动
- **状态**：对神体和未来赵宋的存在一无所知。觉醒时只知道自己突然有了异常能力。
- **认知弧线**：从"我有一个奇怪的力量" → "我脑子里似乎有另一个人" → "那个人是未来的我" → "他为什么回来"

### 赵宋（未来·神体寄附意识）

- **性格**：年轻赵宋的底色 + 80年岁月的层叠——冷冽、沉着、言语诱惑、极重戒心、神经质
- **外貌**：在赵宋脑海中的形象与年轻赵宋一模一样（不是中年面孔）
- **目的**：
  - 终极：分离纠缠的两个宇宙（未来两宇宙纠缠导致了巨大灾难）
  - 过程：引导年轻赵宋成为"自己"
  - 次要：报仇、挽回遗憾
- **处境**：残缺神体 + 零人体 = 每用一次力量就离消散更近。但消散的过程反而让他对神体的控制权趋于完整——一个残酷的交换。
- **生命历程**：在原始时间线上活了约80年，生命有所延长但到最后也只是中年样貌。

## 世界规则

### 已明确定义

1. **双宇宙融合（The Convergence）**：宇宙A（现实）与宇宙B（异质）渐进融合，超然物质（Hyper-Substance）渗入现实。
2. **对称宇宙论**：A/B宇宙无高低之分，只有"常数差异"。A看B是疯狂幻梦，B看A是物质监牢。
3. **逻辑污染**：修行 = 被B宇宙污染。力量的本质是现实逻辑被异宇宙覆写。
4. **力量两阶路径**：
   - Path A（物质态）：超然物质附着肉体，元素操控，仍属物理范畴
   - Path B（概念态）：超然物质相变为概念核心，绕过物理直接改写逻辑
5. **现实抚平机制**：宇宙A的免疫反应。干扰越大回弹越剧烈，可抹杀载体。
6. **双重消耗律**：概念核心不可再生地磨损 + 对肉体/精神的排异消耗。
7. **神体 = 概念核心的表现形式 = B宇宙形态**：人体是A宇宙形态。完整个体需要两者平衡，偏向任何一方都有隐患。
8. **污染程度与信息解锁正相关**：被B宇宙渗透越深，能理解/获取的B宇宙逻辑越多。
9. **神体可独立存在**：未来赵宋的神体独立附着于年轻赵宋。
10. **神体控制权的消散转移**：未来赵宋消散 → 神体控制权完整化 → 归属年轻赵宋。

## 叙事结构线索

故事暗示了 **"层层揭面 + 倒计时递减 + 遗产继承"** 的三轨结构：

1. **揭面轨道**：赵宋的真实能力层级、脑中的声音、回来的真实目的——逐步向读者和角色揭示
2. **倒计时轨道**：神体持续消耗，未来赵宋在消散。每次动用力量都推近终点
3. **遗产轨道**：未来赵宋能传递多少信息、年轻赵宋能理解和继承多少，决定终极任务成败

冲突升级的自然台阶：
1. **伪装期**：以"空间系新人"身份在中阶小团体中生存
2. **暴露期**：能力表现与"超凡者"人设逐渐对不上号，引发怀疑
3. **猎杀期**：多方势力入场，赵宋的"异常"成为争夺焦点
4. **真相期**：概念者/时间回溯者的身份部分曝光，格局重洗
5. **抉择期**：神体接近耗尽，赵宋面对最终选择——是否能完成未来自己的遗志
