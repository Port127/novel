# 任务

查看时间线事件。

## 前置检查

1. 读取 `.current.yaml` 获取 `current_path`

## 输入参数

- `$0` (range): 可选范围，如 `第5-10天`、`第1章-第5章`

支持的筛选：
- 时间范围：`第5-10天`
- 章节范围：`--chapter 1-5`
- 角色筛选：`--character 张三`
- 地点筛选：`--location 青云宗`

## 执行步骤

### 1. 读取时间线

读取 `{current_path}/timeline/main.yaml`。

### 2. 应用筛选

根据参数过滤事件。

### 3. 格式化输出

## 输出格式

```
📅 时间线：{{范围}}

---

## 第5天
- 🌅 清晨：张三离开山村 → 第3章
- 🌞 上午：张三抵达青云宗 → 第4章
- 🌙 夜晚：张三通过入门测试 → 第5章

## 第6天
- 🌞 全天：张三开始修炼基础功法 → 第6章
- 🌙 夜晚：张三遇见赵六 → 第7章

---

📊 统计：
   时间范围：{{start}} ~ {{end}}
   事件数量：{{count}}
   涉及角色：{{characters}}

💡 添加事件：/timeline-add [时间] [事件]
```

## 注意事项

- 时间格式统一
- 显示章节关联
- 支持多维度筛选
