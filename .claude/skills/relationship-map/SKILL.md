# 任务

生成角色关系图谱。

## 前置检查

1. 读取 `.current.yaml` 获取 `current_path`

## 输入参数

- `$0` (character): 可选，指定查看某角色的关系网

## 执行步骤

### 1. 获取角色列表

从 `state.yaml` 读取 characters 列表。

### 2. 提取关系信息

遍历所有角色卡片，提取「人物关系」部分。

### 3. 构建关系图

以文本形式可视化关系网络。

## 输出格式

**全局视图：**
```
📊 角色关系图谱

┌─────────────────────────────────────────┐
│              张三 (主角)                 │
│                    │                     │
│     ┌──────────────┼──────────────┐     │
│     │              │              │     │
│   师兄弟         仇敌          师徒       │
│     │              │              │     │
│   李四          王五           赵六      │
│  (配角)        (反派)         (女主)     │
└─────────────────────────────────────────┘

关系详情：
张三 ←→ 李四：师兄弟
张三 ←→ 王五：仇敌（张三杀了王五父亲）
张三 ←→ 赵六：师徒（赵六是师父）

💡 使用 /relationship-add 添加更多关系
```

**单角色视图：**
```
📊 张三的关系网

张三 (主角)
├── 师兄弟 → 李四 (配角)
├── 仇敌 → 王五 (反派)
│   └── 张三杀了王五父亲
├── 师徒 → 赵六 (女主)
│   └── 赵六是师父
└── 情敌 → 钱七 (配角)
```

## 注意事项

- 关系复杂时分组显示
- 可导出为图片（未来功能）
- 支持按角色筛选
