# novel-kpi 示例

## 示例 1：最近 30 天 KPI

输入：

```bash
/novel-kpi 最近30天
```

输出（节选）：

```markdown
📈 项目KPI：《仙途》（最近30天）

## 产能指标
- 连续更新率：73%
- 完稿率：41%
- 返工率：22%

## 风险治理指标
- 借鉴风险消解率：67%
- AI痕迹消解率：58%
- 关系冲突修复率：80%

## 结论
- 项目状态：可控
- 最优先动作：先将 ch018 从 revise 推进到 final，并完成 ch017 的AI痕迹修复
```

## 示例 2：章节范围 KPI

输入：

```bash
/novel-kpi ch001-ch015
```

输出要点：

- 范围内完稿率与返工率
- 风险消解是否跟上章节推进
- 若数据缺失，明确提示缺失项与补录建议
