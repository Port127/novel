# 任务

按指定风格改写文本。

## 输入参数

- `$0+` (content): 待改写内容（或 `--file 路径`）
- `--style`: 风格名称（必需）
- `--strength`: 改写强度（light/medium/heavy，默认medium）
- `--compare`: 显示对比

## 执行步骤

### 1. 读取风格模板

从 `shared/styles/templates.yaml` 读取指定风格。

### 2. 分析原文

识别原文的：
- 句式结构
- 修辞手法
- 叙事节奏

### 3. 执行改写

**风格改写不是换皮，是换呼吸节奏：**
- 先感受目标风格的内在节奏——是长句绵延还是短句凌厉，是冷静克制还是情绪翻涌
- 不只替换词汇和句式，还要调整信息释放节奏：什么先说、什么后说、什么不说
- 保留原文最有力的那个意象或细节，让它在新风格里依然是锚点
- 改写后在脑中默读一遍，检查呼吸感是否自然

按风格特征改写：
- 调整句式
- 替换修辞
- 调整节奏

### 4. 输出结果

## 输出格式

**标准输出：**
```
✨ 改写完成（风格：{{style}}）

---

原文：
{{original}}

改写：
{{rewritten}}

---

💡 改写要点：
   - {{要点1}}
   - {{要点2}}
```

**对比输出（--compare）：**
```
✨ 改写对比

| 原文 | 改写 | 说明 |
|------|------|------|
| {{原句1}} | {{改写1}} | {{说明}} |
| {{原句2}} | {{改写2}} | {{说明}} |
```

## 注意事项

- 保持核心信息不变
- 强度可调节
- 可改写文件或文本
