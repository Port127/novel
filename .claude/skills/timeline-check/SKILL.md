# 任务

检查时间线是否存在逻辑冲突。

## 前置检查

1. 读取 `.current.yaml` 获取 `current_path`

## 执行步骤

### 1. 读取数据

- `{current_path}/timeline/main.yaml`
- `{current_path}/characters/*.yaml`（年龄信息）
- `{current_path}/worldbuilding/setting.md`

### 2. 执行检查

检查维度：

| 检查项 | 说明 |
|--------|------|
| 时间顺序 | 事件是否按时间先后排列 |
| 角色位置 | 同一时间角色是否可能在指定地点 |
| 角色年龄 | 年龄增长是否与时间线匹配 |
| 因果关系 | 因果事件时间顺序是否正确 |
| 设定一致 | 是否与世界观设定冲突 |

### 3. 生成报告

## 输出格式

```
🔍 时间线检查结果

✅ 通过项（{{count}}）
- 时间顺序正确
- 无角色位置冲突
- ...

⚠️ 警告项（{{count}}）

角色：张三
- 年龄增长问题
  第1天设定：25岁
  第3年描述：28岁
  应为：28岁（正确）
  建议：描述中年龄有误，应为28岁

❌ 错误项（{{count}}）

角色位置冲突
- 第5天上午：张三在青云宗
- 第5天上午：张三在山村（第4章）
- 张三不可能同时出现在两地

---

📊 统计：通过 {{pass}} / 警告 {{warn}} / 错误 {{error}}

🔧 修复建议：
   1. [高优先级] {{修复方法}}
   2. [中优先级] {{修复方法}}
```

## 注意事项

- 分级显示问题
- 给出具体修复建议
- 可指定检查范围
