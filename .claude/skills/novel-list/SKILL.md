# 任务

列出所有已创建的小说项目。

## 执行步骤

### 1. 读取项目列表

从 `.projects.yaml` 读取所有项目。

### 2. 读取各项目简要状态

对每个项目，读取其 `.novel/state.yaml` 获取基本信息。

### 3. 标记当前项目

从 `.current.yaml` 获取当前工作项目，在列表中标记。

## 输出格式

```
📚 所有项目（共 {{count}} 个）

┌─────┬──────────┬────────┬──────────┬────────┐
│ 状态 │ 书名      │ 类型    │ 角色     │ 章节   │
├─────┼──────────┼────────┼──────────┼────────┤
│ 👉  │ 《仙途》  │ 仙侠   │ 5个      │ 15章   │
│     │ 《都市》  │ 都市   │ 3个      │ 8章    │
│ ⏸️  │ 《悬疑》  │ 悬疑   │ 7个      │ 20章   │
└─────┴──────────┴────────┴──────────┴────────┘

👉 = 当前项目   ⏸️ = 暂停   ✅ = 已完结

切换项目：/novel-switch [书名]
新建项目：/novel-init [书名] [类型]
```

## 注意事项

- 按最后更新时间排序
- 显示项目状态（活跃/暂停/完结）
- 如果没有项目，提示创建
