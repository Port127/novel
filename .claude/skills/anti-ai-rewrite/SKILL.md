# 任务

按等级执行去 AI 感改写。

## 前置检查

1. 读取 `.current.yaml` 获取 `current_path`
2. 按 [章节自动推断协议](_protocols/chapter-auto-inference.md) 确定目标章节
3. 读取 `{current_path}/chapters/$0.md`
4. 读取 `shared/styles/anti_ai_rules.yaml`
5. 读取本章出场角色的 `speech_pattern`（对白改写必需）
6. 建议先运行 `/anti-ai-check $0`

## 输入参数

- `$0` (chapter_id): 章节ID
- `--level`: 改写强度
  - `1` 轻改（句式和词汇）
  - `2` 中改（节奏与段落组织）
  - `3` 深改（叙事视角微调但不改事实）

## 执行步骤

### 1. 读取检测报告（如存在）

优先读取 `{current_path}/quality/ai_trace_report.yaml` 中该章节最近一次检测结果，按维度定位问题段落。若报告不存在，先内部快速扫描。

### 2. 按维度分策略处理

**改写目标不是"不像 AI"，是"像一个有审美的人写的"：**
- 替换套话时，不要只找同义词——找一个只有这个角色、这个场景才会出现的表达
- 瘦身比喻时，留下来的那个比喻要精准到让人记住
- 压缩描写时，用一个绝佳细节替代三段平庸描写
- 口语化对白时，想象这个人真的站在你面前说话——他的语速、停顿、吞掉的半句话

#### A. 套话清洗
- 删除或替换命中 `phrase_watchlist` 的套话
- "这一刻"→ 删除或换成具体时间锚点
- "不由得"→ 直接去掉，让动作本身说话
- "仿佛在诉说着"→ 砍掉，换成具体的声音/画面

#### B. 比喻瘦身
- 每千字保留不超过 `metaphor.max_per_1000_chars` 个比喻
- **砍掉策略**：空泛比喻（"如同xxx一般"）直接删，用具体感官细节替代
- **升级策略**：保留的比喻要贴合角色认知——武夫用兵器/伤口做比喻，商人用买卖做比喻
- 连续比喻只留最精准的一个，其余改为白描

#### C. 描写压缩
- 超过 `max_consecutive_desc_paragraphs` 的连续描写段，穿插角色动作或反应打断
- 纯氛围描写要绑定叙事功能：这段描写是为了制造什么情绪？答不上来就砍
- 外貌描写不超过 2 句，且必须通过其他角色的眼睛看（不要上帝视角列清单）

#### D. 对白口语化
- 按 [名字解析协议](_protocols/name-resolution.md) §1a 保留作者已用的所有称呼形式，不替换为正式名
- 读取对白角色的 `speech_pattern`（tone / sentence_style / profanity_level / catchphrase / verbal_tics）
- 把书面语对白改成符合角色身份的口语：
  - 粗人说粗话，不说"你这样做不妥"而说"你他妈搞什么"
  - 少年不说"我理解你的苦衷"而说"行吧，你说的算"
  - 权贵不说大白话，用委婉和暗示
- 去掉通用句式（"我明白了""你说得对"），换成角色特有的反应方式
- 插入角色 `catchphrase` 和 `verbal_tics`
- 参照 `taboo_words` 确保角色不会说不该说的话

#### E. 句式去重
- 连续相同句首（"他...他...他..."）打散，换主语或换句式
- 排比超过 3 句的压缩为 2 句或拆开

#### F. 转折去机械化
- "然而"出现超过 2 次的章节，替换为具体的转折动作
- "就在这时"换成实际发生的事件直接切入

### 3. 分级控制

| 等级 | 范围 | 说明 |
|------|------|------|
| L1 轻改 | 套话清洗 + 句式去重 | 只动词汇和句式，不动段落结构 |
| L2 中改 | L1 + 比喻瘦身 + 描写压缩 + 对白口语化 | 动段落节奏，可能合并/删除段落 |
| L3 深改 | L2 + 叙事视角微调 | 可调整叙述切入角度，但不改事实和顺序 |

### 4. 输出改写版本

逐段输出，每段附：原文 → 改写 → 修改原因（一句话）。

改写后覆盖写入 `{current_path}/chapters/$0.md` 的正文部分。

## 输出格式

```
去AI感改写完成：$0

改写强度：L{{level}}
处理片段：{{count}}处

按维度统计：
  套话清洗    {{count_A}}处
  比喻瘦身    {{count_B}}处
  描写压缩    {{count_C}}处
  对白口语化  {{count_D}}处
  句式去重    {{count_E}}处
  转折去机械  {{count_F}}处

关键改动示例：
1. [比喻瘦身] "她的眼睛如同深邃的湖水一般" → "她盯着他，没眨眼"
2. [对白口语化] "我理解你的处境" → "行，你说咋办"
3. ...

下一步：
   /anti-ai-check $0    验证改写效果
```

## 注意事项

- 不删除关键伏笔
- 不改变人物立场、事件顺序、因果关系
- 改写后建议再次运行 `/anti-ai-check` 对比分数变化
- 对白改写必须参照角色 `speech_pattern`，没有语言画像的角色先提示补充
- 比喻瘦身不是消灭比喻，而是消灭"AI味比喻"——留下贴合角色认知的精准比喻
