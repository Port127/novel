# project-weekly-report 示例

## 示例 1：双视角周报

输入：

```bash
/project-weekly-report 最近7天 --view both
```

输出（节选）：

```markdown
📆 项目周报（管理者版）：《仙途》（最近7天）

## 产能
- 新增章节：3
- 状态推进：2
- 字数完成率：86%

## 风险
- 关系演进风险：1
- 借鉴高风险：1
- AI痕迹高风险：2

## 下周重点
1. 优先修复 ch012 AI痕迹高风险段落
2. 完成 ch013 从 draft 到 revise 推进
3. 补齐 ch010 的关系桥接事件

---

🖋️ 创作周报（作者版）：《仙途》（最近7天）

## 本周亮点章节
- ch011：反转节奏稳定，收尾钩子有效

## 待打磨章节
- ch012：情绪描写重复，人物对白区分度不足
```

## 示例 2：仅管理者视角

输入：

```bash
/project-weekly-report ch001-ch020 --view manager
```

输出要点：

- 章节推进漏斗（idea→published）
- 风险章节列表（高风险优先）
- 下周行动优先级（高/中）
