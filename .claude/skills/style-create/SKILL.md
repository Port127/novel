# 任务

创建新的写作风格模板。

## 输入参数

- `$0` (name): 风格名称（`--from-chapters` 模式下可省略，系统自动命名）
- `$1+` (features): 风格特征描述

可选：
- `--from 素材名` 从素材提取
- `--sample "文本"` 从样本提取
- `--from-chapters [范围]` 从已写章节中提炼作者真实写作风格（见步骤 0）

## 执行步骤

### 0. 从章节提炼模式（--from-chapters）

当指定 `--from-chapters` 时，**从作者实际写出的章节中提炼风格**，而非让作者描述风格。

**这是最准确的风格建立方式——提炼的是作者真实的写作习惯，不是理想中的写法。**

1. **确定分析范围**
   - `--from-chapters ch001-ch005`：分析指定范围
   - `--from-chapters`（无范围）：分析所有状态为 `draft` 及以上的章节
   - 若可分析的章节少于 2 章，提示"样本不足，建议至少 2 章草稿后再提炼风格"

2. **读取章节正文**
   - 只读取`## 正文草稿`区域，跳过场景大纲和写作备忘

3. **分析以下维度**（每条附原文例句）

   | 维度 | 分析内容 |
   |---|---|
   | 句式习惯 | 平均句长、短句比例、是否偏好断句、复合句结构 |
   | 段落节奏 | 对话/叙述/描写的分布比例、段落长度偏好 |
   | 修辞倾向 | 实际用到的比喻类型、意象偏好、是否回避某类修辞 |
   | 对白风格 | 对白占比、角色间的语气差异、是否有台词外动作习惯 |
   | 心理描写 | 内心独白频率和方式（嵌入式/独立段/无） |
   | 感官偏好 | 视觉/听觉/触觉/嗅觉哪种最常出现 |
   | 转折方式 | 场景切换、时间跳跃、章尾处理的惯用手法 |

4. **输出提炼报告，等待确认**
   - 展示每个维度的分析结论和原文依据
   - 询问用户是否保存为风格模板，以及模板名称

5. **确认后写入**
   - 进入步骤 2-3 生成并保存模板

### 1. 解析风格定义（手动模式）

从用户描述或素材中提取风格特征。

### 2. 生成风格模板

```yaml
- name: $0
  tags: [{{标签}}]
  features:
    sentence:
      - {{句式特点1}}
      - {{句式特点2}}
    rhetoric:
      - {{修辞特点}}
    dialogue:
      - {{对话特点}}
    rhythm:
      - {{节奏特点}}
  examples:
    - "{{示例1}}"
    - "{{示例2}}"
  use_cases:
    - {{适用场景1}}
```

### 3. 保存到风格库

追加写入 `shared/styles/templates.yaml`。

## 输出格式

```
✅ 风格模板已创建

🎨 名称：$0
🏷️ 标签：{{tags}}

## 特征摘要
- 句式：{{sentence_feature}}
- 对话：{{dialogue_feature}}
- 节奏：{{rhythm_feature}}

## 示例
> {{example}}

📁 已保存：shared/styles/templates.yaml

💡 应用风格：/rewrite --style $0 [内容]
```

## 注意事项

- 风格名称唯一
- 特征要具体可操作
- 提供示例
